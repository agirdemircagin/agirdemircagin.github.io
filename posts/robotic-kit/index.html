Robotic Kit

I designed a new robotic kit which include Arduino UNO, Bluetooth, 4 various sensor. This kit completely printable with 3d printer. Also, this kit include a tray system; so students can be made themselves design in tune with tray gaps.

This kits consist of 11 part.

Specifications

  • Arduino UNO R3
  • Settled Bluetooth Module
  • Distance Sensor (Ultrasonic)
  • Unlimited sensor connectable with settled breadboard
  • Buzzer, RGB Led, Tempature and Humid Sensor
